## PyPI Distribution

### Added

- **PyPI Package Published**: Install with `pipx install nexus-ai-cli`
- **Automated Releases**: GitHub Actions workflows for version management
- **Semantic Versioning**: Automated version bumping via conventional commits
- **Trusted Publishing**: Secure PyPI publishing without API tokens

### Installation

#### Quick Install (Recommended)

```bash
pipx install nexus-ai-cli
```

#### Alternative: Using pip

```bash
pip install nexus-ai-cli
```

#### Update

```bash
pipx upgrade nexus-ai-cli
# or
pip install --upgrade nexus-ai-cli
```

### What's New in v0.2.0

This is the first version published to PyPI. Users can now install Nexus AI with a single command.

### System Requirements

- Python 3.8 or higher
- API keys for at least one AI provider (OpenAI, Anthropic, or OpenRouter)
- Ollama users: Ollama server running at http://localhost:11434
